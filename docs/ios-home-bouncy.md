## Description

Scan to try on your device

### Spring Customizer

- Mass: Controls how "heavy" the animation feels
- Damping: Determines how quickly it settles
- Stiffness: Affects how snappy and responsive it is

- Opening: Adjust how the icons expand outward
- Closing: Fine-tune how they bounce back

### Play with the sliders

- Pull down on the customizer to dismiss it
- Tap anywhere to trigger the animation
- Watch values update in real-time as you drag the sliders
