# Blurred Bottom Bar

> **Live Demo**: [View on reactiive.io](https://reactiive.io/demos/blurred-bottom-bar)
> **Source Code**: [/src/animations/blurred-bottom-bar](/src/animations/blurred-bottom-bar)

## What's This About?

This component creates a modern bottom navigation bar that incorporates blur effects for a glass-like appearance. It includes multiple demo screens that showcase different visual effects and animations, all accessible through the bottom bar navigation.

## Technical Implementation

The component uses React Navigation's bottom tab navigator and implements:

- Custom tab bar with blur effects
- Screen freezing for performance optimization
- Multiple demo screens:Home screenScrollable image listBlur effect demonstrationAnimated gradient display
- Home screen
- Scrollable image list
- Blur effect demonstration
- Animated gradient display

## Usage

```tsx
// The component integrates with React Navigation or Expo Router<BlurredBottomBar />
```
